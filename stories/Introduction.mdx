# TDM Design System Web

Design system berbasis Nuxt Layer dengan komponen UI yang siap pakai, built dengan shadcn-vue dan Tailwind CSS 4.

## Tentang Design System Ini

TDM Design System Web adalah komponen library yang powerful dan fleksibel yang dirancang untuk mempercepat development aplikasi web dengan Nuxt.js. Dibangun di atas fondasi solid dari shadcn-vue dan Tailwind CSS 4, design system ini menyediakan:

- **50+ komponen UI siap pakai** yang telah dioptimasi dan tested
- **Auto-import components** untuk developer experience yang seamless
- **Design tokens konsisten** untuk branding yang unified
- **TypeScript support** untuk type safety
- **Accessible components** mengikuti best practices
- **Customizable styling** dengan Tailwind CSS 4
