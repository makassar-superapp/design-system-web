# TDM Design System Web

A Nuxt Layer-based design system with ready-to-use UI components, built with shadcn-vue and Tailwind CSS 4.

## About This Design System

TDM Design System Web is a powerful and flexible component library designed to accelerate web application development with Nuxt.js. Built on the solid foundation of shadcn-vue and Tailwind CSS 4, this design system provides:

- **50+ ready-to-use UI components** that are optimized and tested
- **Auto-import components** for seamless developer experience
- **Consistent design tokens** for unified branding
- **TypeScript support** for type safety
- **Accessible components** following best practices
- **Customizable styling** with Tailwind CSS 4
